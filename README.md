## Quick Start

1. Clone the repository:
   ```
   git clone https://github.com/your-username/codecrate.git
   ```

2. Install dependencies:
   ```
   cd codecrate
   npm install
   ```

3. Run the development server:
   ```
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
